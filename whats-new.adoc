=== 适用于VM 1.0的BlueXP备份和恢复现已正式发布

您可以通过对VM使用BlueXP备份和恢复来保护虚拟机上的数据。您可以在Amazon FSx上备份数据存储库并将虚拟机还原到其原始位置。

此解决方案专用于VMware Cloud on AWS和Amazon FSx for NetApp ONTAP上的虚拟机和数据存储库。
